Timestamp Microservice

Example usage:

You can use one of the following formats as the project endpoint:

Natural date format:

.../2017-12-02

... or Unix-Timestamp(in milliseconds):

.../1480636850000

Example output:

{
  "unix": 1512172800000,
  "utc": "Sat, 02 Dec 2017 00:00:00 GMT"
}
